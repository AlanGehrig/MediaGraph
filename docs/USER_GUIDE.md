# MediaGraph 摄影师使用指南

## 简介

MediaGraph 是一款专为摄影师设计的 AI 影像知识图谱系统。它能够：

- 🔍 **自然语言搜索** - 用日常语言搜索你的所有素材
- 👤 **人脸聚类** - 自动识别照片中的人物
- 🗂️ **知识图谱** - 建立人物-时间-地点的关系网络
- 🎬 **视频解析** - 自动分析视频内容
- 📊 **多维分析** - 场景、情绪、光线、构图全维度提取

## 快速开始

### 1. 首次使用

1. 启动服务:
   ```powershell
   .\start_media_graph.bat
   ```

2. 打开浏览器访问:
   - 前端界面: http://localhost:3000
   - API文档: http://localhost:8000/docs

### 2. 扫描素材

1. 在配置文件中设置素材路径:
   ```yaml
   media:
     scan_paths:
       - E:/我的照片
       - E:/工作素材
   ```

2. 点击界面中的"扫描"按钮，或使用API:
   ```bash
   curl -X POST http://localhost:8000/api/media/scan
   ```

### 3. 等待AI解析

系统会自动对新扫描的素材进行 AI 解析，提取:
- 场景信息 (海边、城市、室内等)
- 主色调 (蓝色、暖色等)
- 光线条件 (自然光、逆光、柔光等)
- 情绪氛围 (开心、浪漫、戏剧性等)
- 构图分析 (三分法、特写、宽幅等)

## 人脸聚类

### 查看人物

1. 进入"人物"页面
2. 系统会自动将相似人脸聚合成组
3. 每个聚类代表一个独立个体

### 命名人物

1. 点击未命名的人物聚类
2. 输入人物名称 (如"张三")
3. 系统会记住这个人的所有照片

### 重新聚类

如果聚类效果不理想，可以:
1. 调整聚类阈值
2. 点击"重新聚类"按钮

## 素材管理

### 查看详情

点击任意素材，查看:
- 文件信息 (尺寸、格式、大小)
- EXIF信息 (相机、镜头、参数)
- AI分析结果
- 关联的人物和地点

### 批量操作

- 批量选择素材
- 批量添加标签
- 批量解析

## 数据导出

### 导出搜索结果

1. 执行搜索
2. 选择要导出的结果
3. 点击"导出"
4. 选择格式 (JSON/CSV)

### 备份图谱

定期运行备份脚本:
```powershell
.\scripts\backup_kg.bat
```

## 隐私保护

### 本地存储

所有数据都存储在本地:
- Neo4j 数据库: `data/neo4j`
- Chroma 向量: `data/chroma`
- 配置文件: `config/`

### 无云端依赖

系统完全离线运行，不上传任何数据到云端。

## 性能优化

### GPU加速

如果有多显卡，可以启用GPU加速:
```yaml
model:
  clip_device: cuda
  insightface_device: cuda
```

### 增量扫描

定期运行扫描只会处理新增文件，不会重复处理已有素材。

## 常见问题

### Q: 搜索结果不准确?
A: 尝试更具体的描述，或使用多个关键词组合。

### Q: 人脸识别失败?
A: 检查照片中人脸是否清晰、遮挡少。

### Q: 视频解析很慢?
A: 视频解析比较耗时，可以限制采样率或分批处理。

### Q: 如何添加新的素材路径?
A: 编辑 `config/env.windows.yaml` 中的 `media.scan_paths`。
