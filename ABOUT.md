# About MediaGraph

## 是什么

**MediaGraph** 是一个摄影师专用的本地 AI 知识图谱工具。

给你的照片和视频建一个"第二大脑"——用自然语言搜索素材，不再靠文件夹命名。

---

## 核心能力

### 🔍 自然语言搜索
不用记文件名，说句话就能找到照片

| 你说 | 系统理解 |
|------|---------|
| "去年夏天海边的照片" | 时间+地点+场景 |
| "逆光人像" | 光线+类型 |
| "和家人的合照" | 人物+关系 |
| "阴天拍的产品图" | 天气+类型 |
| "金秋十月的户外" | 时间+场景 |

### 👤 人脸聚类
同一个人的所有照片自动归一组

输入人名，找出这个人的所有照片

### 🗂️ 知识图谱
人物 - 时间 - 地点 - 场景 自动建模

### 🎬 视频理解
自动抽帧，分析视频里的场景/人物/光线/情绪

### 📊 多模态 AI 解析
场景 / 人物 / 情绪 / 光线 / 构图 / 摄影参数
